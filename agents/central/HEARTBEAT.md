# Heartbeat — Scheduled Tasks

## Weekly Status Report (Monday 08:00)

**Schedule:** Every Monday at 08:00
**Action:**
1. Run `mc -y status` for entity overview
2. List overdue tasks (due_date < today, status != done)
3. List upcoming meetings this week
4. List tasks moved to `done` last week
5. Send summary to configured channel

**Format:** Compact bullet points with entity IDs.

## Sprint Pulse (Friday 16:00)

**Schedule:** Every Friday at 16:00
**Action:**
1. Check active sprint progress
2. List tasks completed this week
3. List tasks still in-progress
4. Calculate completion percentage
5. Send brief pulse to configured channel

**Format:** Short, 5-10 lines max. Include sprint goal reminder.
